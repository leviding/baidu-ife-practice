## 复习要点

今天代码部分其实比较简单，主要是多尝试，学习之后，回顾以下自己是否已经掌握以下概念：

1. 背景，边框，列表，链接相关属性
2. CSS 各种选择器的概念，使用方法及使用场景
3. CSS 选择器的优先级
