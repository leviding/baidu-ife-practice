## 复习要点

学习之后，回顾以下自己是否已经掌握以下概念：

1. 盒模型的概念
2. inline、block 和 inline-block 的概念
3. 内外边距，宽度，高度，box-sizing 等属性
4. 浮动，清除浮动
5. 如何使用浮动进行布局
