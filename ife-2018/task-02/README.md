## 复习要点

今天代码部分其实比较简单，主要是多尝试，学习之后，回顾以下自己是否已经掌握以下概念：

1. 什么是 CSS，CSS 是如何工作的
2. CSS 的基本语法是怎样的
3. CSS 选择器是什么概念，简单选择器和属性选择器是什么
4. 文本样式都有哪些相关属性，对应哪些值
